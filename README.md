# 🎓 Edunova - Frontend Developer

Dobrodošli u moj repozitorij za učenje! Ovdje dokumentiram svoj napredak tijekom **Frontend Developer tečaja**. Svaka mapa u ovom repozitoriju predstavlja određenu lekciju i/ili zadatak.

### 📚 Koje tehnologije ću pokriti u ovom repou:
* **Struktura:** Semantički HTML5.
* **Stilizacija:** CSS3 (Flexbox, Grid, Responsive Design), Bootstrap v5, Foundation CSS.
* **Logika:** JavaScript osnove i DOM manipulacija (algoritamski zadaci).
* **Framework:** React.js (Hooks, Components, Props, State management).
* **Alati:** Git, GitHub, FileZilla.

### 🛠️ Kako koristiti:
Svaki folder je zaseban zadatak (lekcija). Rezultati zadataka se mogu vidjeti i na domeni: [edunova3.origo.hr](https://edunova3.origo.hr/) , u nastavku url-a se samo stavi putanja do datoteke `index.html` određenog zadatka.
